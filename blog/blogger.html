What it does:

The application creates a GUI which simplifies the process of writing a blog for obryan.xyz.

How does it do it?

The application uses Tkinter, one of Python's built in libraries for the GUI. Within Tkinter, blogger uses the grid system within a scrollable frame. There is then an "Add Section" button which adds a header field and body field to the scrollable frame. There are an unlimited amount of sections that you can add and they work seamlessly.
